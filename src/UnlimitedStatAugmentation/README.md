# Unlimited Stat Augmentation Gems

This optional OpenDereth server mod removes ACE's shared limit of ten innate-stat augmentation gems. While enabled, characters may continue using Strength, Endurance, Coordination, Quickness, Focus, and Self gems until each affected innate stat reaches 100.

The mod does not make the gems free, create gems, change their experience cost, or alter any other augmentation family. Skill specialization, resistance, pack-slot, burden, critical, damage, and all other non-stat augmentation limits remain stock ACE behavior.

The server's attribute safety cap is forced on while the mod is active. A stat gem still adds at most five points and cannot raise an innate stat above 100.

Turning the mod off restores the normal shared usage limit after a server restart. Stat increases already saved on characters remain; the mod does not reduce or rewrite existing character attributes.

This package is a preview until its complete acquisition and boundary behavior has been tested in game. Back up `%LOCALAPPDATA%\OpenDereth` before testing progression-changing mods.
