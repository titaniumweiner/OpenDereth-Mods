# Unlimited Skill Specializations

This optional OpenDereth server mod removes ACE's normal limit of 70 total credits invested in specialized skills.

The mod changes only the total specialization ceiling checked when a Gem of Enlightenment is used. The target skill must still be trained, the character must still have enough available skill credits, and the normal specialization credit cost is still deducted. Skill ranks, experience, heritage-adjusted costs, skill-lowering gems, and specialization augmentations keep their stock ACE behavior. ACE's administrator skill-credit verifier also accepts over-cap specializations while the mod is enabled instead of offering to reset them.

Specializations above the normal 70-credit ceiling are saved character data. Turning the mod off prevents additional over-cap specializations after a server restart, but it does not automatically unspecialize skills or refund credits. Use Gems of Forgetfulness normally if you want to lower skills before disabling the mod.

This package is a preview until specialization, saving, restart, and removal have been tested thoroughly in game. Back up `%LOCALAPPDATA%\OpenDereth` before testing progression-changing mods.
